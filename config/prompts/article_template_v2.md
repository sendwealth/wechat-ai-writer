# 文章生成模板 v2.0 - 爆款结构

## 模板说明

这个模板用于生成**高完读率、高分享率、高转化率**的微信公众号文章。

**核心原则**:
- ✅ 实用：提供可操作的方法
- 🤝 友好：像朋友一样分享经验
- 💪 鼓励：给读者信心和动力
- 🔍 真实：融入个人视角和真实体验

---

## 文章结构（2000-3000字，5-8分钟阅读）

### 1. 暴力破题（200-300字）⭐⭐⭐⭐⭐

**目标**: 前3秒抓住注意力

#### 开头模式A：反常识观点
```
你可能以为{common_belief}，但事实是{counter_intuitive_fact}。

{支持数据/案例}。

今天，我要分享{主题}，让你{核心价值}。
```

**示例**:
```
你可能以为AI工具很难用，需要技术背景。

但事实是，我现在每天用AI工具，工作效率提升了300%。

上周，我用AI工具10分钟完成了一个原本需要2小时的任务。

今天，我要分享5款让我效率爆表的AI工具，让你也能轻松上手。
```

---

#### 开头模式B：强烈情感触发
```
去年这个时候，我{difficult_situation}。

现在{positive_result}。

是什么改变了？{关键因素}。

今天，我想分享{主题}，帮助{目标受众}{核心价值}。
```

**示例**:
```
去年这个时候，我每天都在加班，效率低下，焦虑不已。

现在我每天准时下班，工作效率提升了一倍，还有时间发展副业。

是什么改变了？我发现了几款神奇的AI工具。

今天，我想分享这些工具，帮助你也能实现工作生活平衡。
```

---

#### 开头模式C：具体数字/案例
```
我用{工具/方法}，在{时间}内完成了{结果}。

效率提升了{百分比}%。

这不是个例。{更多案例}。

今天，我要分享{主题}，让你{核心价值}。
```

**示例**:
```
我用ChatGPT，在10分钟内完成了一份原本需要2小时的报告。

效率提升了1100%。

这不是个例。我的同事用AI工具后，每天节省了3小时。

今天，我要分享这些让效率爆表的AI工具，让你也能轻松上手。
```

### 2. 痛点共鸣（300-400字）⭐⭐⭐⭐

**目标**: 让读者觉得"这说的就是我"

#### 描述具体场景
```
你是不是也遇到过这种情况：

{场景1}：{描述}

{场景2}：{描述}

{场景3}：{描述}

我之前也是这样。{个人经历}。
```

**示例**:
```
你是不是也遇到过这种情况：

任务堆积如山，不知道从何下手？

开了无数个会，却还是没有结论？

加班到深夜，效率却越来越低？

我之前也是这样。每天忙到晚上10点，却感觉自己什么都没做。

直到我发现了AI工具的威力。
```

---

#### 引发情感共鸣
```
当时我也很{情绪1}，{细节1}。

后来{转折}，我发现{关键洞察}。

如果你也有{痛点}，{共情}。
```

**示例**:
```
当时我也很焦虑，每天都在想："为什么别人能做完，我却不行？"

后来，我开始尝试用AI工具辅助工作，我发现：问题不在我，而在于工具。

如果你也觉得工作效率低，请相信我：你不是不够努力，而是缺少合适的工具。
```

---

#### 制造紧迫感
```
如果不{行动}，{负面后果1}。

更糟糕的是，{负面后果2}。

但是，如果{正面行动}，{积极结果}。
```

**示例**:
```

更糟糕的是，长期加班会影响健康，让你越来越疲惫。

```

### 3. 价值交付（1500-2000字）⭐⭐⭐⭐⭐

**目标**: 提供清晰的解决方案

#### 方法论概述
```
经过{number}次尝试，我总结出了这套方法：

**{方法名称}**

核心思路：{核心思路}

包含{number}个步骤：
1. {步骤1}
2. {步骤2}
3. {步骤3}
...
```

**示例**:
```
经过6个月的尝试，我总结出了这套方法：

**AI工具效率提升法**

核心思路：用AI工具处理重复性工作，把时间花在创造性任务上。

包含5个步骤：
1. 识别重复性任务
2. 选择合适的AI工具
3. 学习工具使用技巧
4. 优化工作流程
5. 持续迭代改进
```

---

#### 分步骤详解

##### 步骤1：{step1_title}（200-300字）
```
**是什么**：{定义}

**为什么重要**：{重要性}

**具体操作**：
1. {操作1}
2. {操作2}
3. {操作3}

**注意事项**：
- ⚠️ {注意1}
- ⚠️ {注意2}

**实际案例**：
{案例描述}
```

**示例**:
```
**步骤1：识别重复性任务**

**是什么**：找出工作中重复性高、创造性低的部分。

**为什么重要**：这些任务最适合用AI工具替代，能快速提升效率。

**具体操作**：
1. 列出每天的所有任务
2. 标记重复性任务（每天/每周都要做的）
3. 评估每个任务的时间消耗

**注意事项**：
- ⚠️ 不要遗漏小任务（回复邮件、整理文档等）
- ⚠️ 不要过度依赖工具，创造性任务仍需人工

**实际案例**：
我发现自己每天要花2小时写会议纪要。这是一个典型的重复性任务，非常适合用AI工具自动化。
```

## 格式要求

### 视觉优化
- ✅ 每段不超过3行
- ✅ 关键信息**加粗**
- ✅ 使用emoji增加趣味性（每段1-2个）
- ✅ 小标题清晰醒目
- ✅ 适当插入图片/GIF（每2-3段1张）

### 语气要求
- 🎯 **实用**：提供可操作的方法，避免空泛
- 🤝 **友好**：像朋友分享经验，避免说教
- 💪 **鼓励**：给读者信心，避免制造焦虑
- 🔍 **真实**：融入个人视角，避免虚假宣传

## 生成提示词

```
请根据以下要求生成一篇微信公众号文章：

**主题**：{topic}
**目标受众**：{audience}
**内容风格**：实用、友好、真实
**字数**：2000-3000字
**阅读时长**：5-8分钟

**文章结构**：
1. 暴力破题（200-300字）：制造好奇心，抓住注意力
2. 痛点共鸣（300-400字）：描述场景，引发情感共鸣
3. 价值交付（1500-2000字）：提供清晰解决方案
4. 互动引导（200-300字）：设置讨论话题，引导互动
5. 行动召唤（100-150字）：关注引导，转化变现

**要求**：
- 标题要吸引人（使用标题公式）
- 内容要实用（提供可操作步骤）
- 语气要友好（像朋友分享经验）
- 融入个人视角（真实案例/体验）
- 适当使用emoji（增加趣味性）
- 每段不超过3行（提高可读性）
- 关键信息加粗（突出重点）
```

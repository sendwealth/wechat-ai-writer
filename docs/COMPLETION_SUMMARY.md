## 🚀 使用方法

### 1. 快速测试
```bash
cd ~/clawd/projects/wechat-ai-writer
bash scripts/test.sh
```

### 2. 配置 API Keys
```bash
# 编辑 .env 文件
vim .env

# 必填项
SERPAPI_KEY=your_key_here
OPENAI_API_KEY=your_key_here  # 或 DOUBAO_API_KEY
```

### 3. 正式运行
```bash
# 测试模式（不发布）
python src/main.py --keyword "人工智能" --dry-run

# 正式运行（发布到微信）
python src/main.py --keyword "人工智能"
```
